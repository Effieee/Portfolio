## Overview

This website was built as a practice project following the Frontend Mentor challenge instructions and starter files. The website contains a desktop and mobile versions and adjustments for several screen sizes.

### The challenge

Users should be able to:

- View the optimal layout for the site depending on their device's screen size
- See hover states for all interactive elements on the page
